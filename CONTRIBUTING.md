# Contributing

Falon Client is currently prepared as a curated public repository.

## Before opening a PR

- Keep the UI and documentation in English.
- Do not include generated keys, vault files, installers, binaries, logs, or local debug data.
- Do not commit files from `dist/`, `node_modules/`, or local audit/probe scripts.
- Keep changes focused and easy to review.

## Local development

```powershell
npm install
npm start
```

## Security

Do not open public issues for security problems or leaked private assets. See `SECURITY.md`.
